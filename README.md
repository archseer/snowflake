# snowflake

Deterministic NixOS configuration for all my devices.

# Setup

```
# Usage: rebuild host {switch|boot|test|iso}
rebuild <host-filename> test
```

## Build an ISO

You can make an ISO and customize it by modifying the [iso](./hosts/iso.nix)
file:

```sh
rebuild iso
```
